## Image Specifications

- **Format**: JPG or PNG
- **Recommended Size**: 
  - Hero images: 1920x600px
  - Card images: 600x400px
  - Logo: 300x300px (square)
  - Thumbnails: 300x300px
- **Optimization**: Compress images for web to ensure fast loading
- **Naming**: Use lowercase with hyphens (kebab-case)

## Notes

- All images should be optimized for web use
- Maintain aspect ratios for consistent display
- Use high-quality images that represent the college professionally
- Ensure all images are properly licensed or owned by the college
